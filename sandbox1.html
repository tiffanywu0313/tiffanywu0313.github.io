UBC Forestry CALP Lab

During my time with the CALP Lab, I built an interactive educational tool to engage citizens with climate change called iCoolkit.



Product:       Responsive Website

Team:             Tiffany W., Doris S., Cindy C., Stephen S.

Role:               Graphic Design, UI Design, Web Dev

Tools:             Photoshop, Illustrator, HTML, CSS, JS

Timeline:     May 2019 - April 2020

Context

CALP stands for Collaborative for Advanced Landscape Planning. When I joined CALP, they had already created an educational tool called Coolkit that aims to engage students and citizens with climate change. What they wanted me to do was transform the paper version into a digital version (iCoolkit) with interactive components.



Process:

When I started transforming the Coolkit into a digital version, I noticed many errors in the Coolkit document itself. So after speaking with my supervisor, we decided to finish editing the Coolkit first then work on the digital version.

I helped with a lot of the grammar correcting and updating graphics. Thus I touched on a lot of tools during this internship, including using photoshop and illustrator to create and update graphics, using indesign to update the format of the document and finally coding it into a website.
